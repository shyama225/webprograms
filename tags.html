TEXT STYLES

HTML(Hyper Text Markup Language ) HTML is the standard markup language for creating Web pagesHTML describes the structure of a Web page HTML consists of a series of elements

HTML attributes provide additional information about HTML elements.

Bold tag is used to make text bold. It is a presentational element that does not provide any semantic meaning to the content.

Italicstag is used to make text italic. Like the "b" tag for bold text, the "i" tag is a presentational element, meaning it changes the appearance of the text without providing any semantic meaning.

underlinetag to underline text. However, the "u" tag was originally used to underline text in earlier versions of HTML, but it was deprecated in HTML5 due to its purely presentational nature.
